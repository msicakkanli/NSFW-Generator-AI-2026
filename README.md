## 📝 How to Make a Good Prompt

Use short, clear words. Start with the character, then add style and scene details.

Example prompt structure:

- Character type
- Hair, face, clothing, pose
- Art style
- Background
- Lighting

Example:

young fantasy warrior, silver hair, leather armor, standing pose, detailed concept art, soft light, forest background

You can also add simple quality words like:

- detailed
- clean lines
- dramatic lighting
- cinematic background
- full body

If the result is not close to what you want, change one part of the prompt at a time.

## 🛠️ Basic Troubleshooting

### The app does not open
- Check that you downloaded the full release file
- Try running it again as administrator
- Make sure your antivirus did not block the file

### The app opens, but generation is slow
- Close other heavy apps
- Lower the image size
- Use fewer steps
- Make sure your GPU drivers are up to date

### You see a black screen or missing images
- Restart the app
- Check that the model files are in the correct folder
- Use a newer graphics driver

### Windows SmartScreen appears
- Select More info
- Then choose Run anyway if you trust the file from the release page

### The app cannot find a model
- Confirm the model file is in the model folder
- Make sure the file name is not changed
- Restart the app after adding the model
